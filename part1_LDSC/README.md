# Running the LDSC Script

This script runs LDSC to analyze genetic correlation between different BMI datasets.

### Input data:
1. **LDSC Path**: Path to the LDSC Python script (e.g., `path/to/ldsc.py`)
2. **Input data directory**: Path to the directory containing the input summary statistics files.
3. **Reference LD directory**: Path to the reference LD files for the EUR population (e.g., `path/to/reference_LD/`).

You need to replace the following paths in the script:
- `LDSC_PATH`
- `INPUT_DIR`
- `REF_LD_DIR`
